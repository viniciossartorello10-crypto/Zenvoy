Como funciona

1. Conte sua viagem
Destino, datas, orçamento e estilo — a IA monta um roteiro completo em segundos.

2. Compare e escolha
Veja voos, hospedagem e passeios com preços reais quando disponíveis.

3. Reserve com segurança
Cada opção leva você direto para o site oficial do parceiro (Booking, Duffel, Airbnb, etc.) para finalizar a reserva — a Zenvoy nunca processa pagamentos diretamente.

Perguntas frequentes

A Zenvoy cobra alguma coisa?
Não. Sempre foi e sempre será gratuito usar a Zenvoy para planejar sua viagem.

Minhas informações de pagamento passam pela Zenvoy?
Não. Toda reserva é finalizada diretamente no site do parceiro (Booking, Duffel, Airbnb, etc.), com segurança deles.

Preciso de uma chave de API para usar?
Sim, uma chave gratuita do Google Gemini é necessária para a IA gerar seu roteiro. Leva menos de 2 minutos para criar.

Posso planejar viagens com vários destinos?
Sim! Use o botão "+ Add another destination" no formulário para adicionar quantas cidades quiser.

Quem somos

A Zenvoy nasceu de uma ideia simples: planejar uma viagem não devia significar abrir dez abas diferentes. Somos uma plataforma de viagem com inteligência artificial que reúne voos, hospedagem, roteiro e recomendações locais num só lugar — pensado para viajantes que querem gastar tempo aproveitando a viagem, não organizando ela.
